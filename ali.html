HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Use tags like <h1>, <p>, <a>, <img> etc.

Example: <p>This is a paragraph.</p>

CSS

CSS (Cascading Style Sheets) styles your web pages. You can change colors, fonts, spacing, and layout.

Example: p { color: red; font-size: 18px; }

JavaScript

JavaScript makes web pages interactive. You can respond to user actions, manipulate HTML, and more.

Example: alert('Hello!');
